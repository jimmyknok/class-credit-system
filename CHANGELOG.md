# Changelog

## [1.3.2] - 2026-03-02

### ✨ New Features

- **批量删除学生**：批量操作模式下新增「批量删除」按钮（右侧红色），二次确认后删除所选学生及其全部积分、兑换历史记录，删除完成后自动退出批量模式
- **学生卡片编辑入口**：每张学生卡片操作区新增 ✏️ 编辑按钮，可直接修改姓名、分组并删除单个学生，无需再从小组管理页进入

### 🔧 Improvements

- **排行榜限制前 10 位**：个人排行和小组排行均只展示积分最高的前 10 名，界面更简洁

---

## [1.3.1] - 2026-03-02

### 🔧 Improvements

- **批量操作按钮文案调整**：「批量加减分」统一改为「批量操作」，更准确反映该模式下支持加分、减分、兑换等多种操作
- **学生卡片简化**：隐藏总积分显示，只展示可用积分，界面更简洁；总积分数据仍在后台保留，不影响历史记录
- **隐藏「按总分」排序按钮**：排序栏仅保留「按可用」和「按姓名」，默认排序改为按可用积分降序

---

## [1.3.0] - 2026-03-02

### ✨ New Features

- **新增「月度报告」导航页**：独立页面展示月度积分增长统计与规则分析
- **积分增长 Top 5**：统计所选时段内加分最多的前 5 名学生，展示增长值及与上期对比（↑↓ 箭头），奖励区域占位显示「🎁 奖励待定」，后续填入实际奖励内容
- **时间范围切换**：支持「本月 / 上月 / 自定义时间段」三种模式，自定义模式可选任意起止日期
- **规则统计柱状图**：正向规则和负向规则分两组柱状图展示触发次数，引入 Chart.js 可视化
- **规则明细表格**：列出所有触发规则的名称、类型（加分/扣分）、触发次数、涉及人数，支持点击表头排序

### 📌 Notes

- 月度增长只统计加分记录，减分和兑换操作不计入增长
- 与上期对比的「上期」时间段长度与当前所选时段等长（如选本月，上期即上月）

## [1.2.0] - 2026-03-01

### ✨ New Features

- **批量兑换奖励**：在学生管理页开启批量模式后，操作栏新增「批量兑换」按钮，可一次性为多名学生发放同一奖励
  - 商品卡片实时显示「X 人积分不足」提示，积分不足时整张灰显，与单人兑换样式逻辑一致
  - 积分不足的学生自动跳过，其余正常扣除，操作完成后 Toast 显示兑换成功人数和跳过人数
  - 库存按实际兑换人数扣减，超出库存时阻止操作并提示
- **小组批量兑换入口**：小组管理页每张小组卡片新增 🛍️「兑换奖励」图标，点击后以该组全体成员为目标，直接进入批量兑换弹窗，无需手动逐一勾选

### 🔄 逻辑调整

- **排行榜改用可用积分计算**：个人排行榜和小组排行榜的排序与展示数值，均从历史累计总分（`totalScore`）改为当前可用积分（`availScore`），兑换奖励后排名会实时变化，更直观反映当前积分持有情况
- **小组管理页总分/均分改用可用积分**：小组卡片展示的总分与均分同步改用 `availScore` 口径，与排行榜保持一致，均分标签加注「可用」以区分含义
- 学生管理页的「按总分」和「按可用」两个排序按钮口径不变，保持独立

## [1.1.1] - 2026-03-01

### 🐛 Bug Fixes

- **修复启动时数据加载失败问题**：将 `Promise.all` 并发请求改为顺序 `await`，解决 Supabase JS SDK 在并发场景下触发的 `DataCloneError: Headers object could not be cloned` 异常，该错误会导致页面白屏、数据无法加载。

## [1.1.0] - 2026-03-01

### 🚀 Features

- **接入 Supabase 云数据库**：数据存储从浏览器本地 `localStorage` 迁移至 Supabase（PostgreSQL），所有数据持久化在云端，刷新页面、更换设备不再丢失数据
- **多端实时同步**：基于 Supabase Realtime 订阅，任意设备的操作（加减分、添加学生、编辑小组等）会实时推送至所有在线设备，无需手动刷新
- **云端启动加载**：首次打开页面时显示加载遮罩，连接云端数据库成功后自动消失，避免数据未就绪时的空白闪烁
- **统一错误处理**：所有数据库操作增加错误捕获，失败时通过 Toast 提示具体错误位置，方便排查

### 🔄 Breaking Changes

- 本地 `localStorage` 数据不会自动迁移，如有旧数据需通过「导出 CSV」备份后重新导入

---

## [1.0.1] - 2026-02-28

### 🐛 Bug Fixes

- **修复编辑小组无响应问题**：将小组编辑弹窗从浏览器原生 `prompt()` 改为自定义模态框。原因是浏览器在用户勾选「不再显示此类提示」后会永久屏蔽页面内所有原生弹窗调用，导致编辑按钮点击后无任何响应。

### ✨ Improvements

- 编辑小组弹窗现支持同时修改**小组名称**和**小组别称**，交互更直观
- 新增小组名称为空、名称重复的输入校验提示，防止误操作

---

## [1.0.0] - 2026-02-28

### 🎉 Initial Release

- 学生管理：批量添加、搜索、排序
- 积分操作：单人加减分、批量加减分、积分明细、撤销记录
- 双积分体系：总积分（历史累计）+ 可用积分（可消费余额）
- 小组管理：添加 / 编辑 / 删除小组，实时积分排行
- 积分规则：自定义正负规则，加减分时快速套用
- 积分商店：自定义商品、兑换消耗积分、库存管理
- 排行榜：个人 / 小组实时排行，金银铜徽章
- 课堂工具：圆盘倒计时器 + 随机点名
- 数据面板：导入导出学生名单、规则配置、兑换记录（CSV / JSON）
- 数据持久化：所有数据本地 localStorage 存储
